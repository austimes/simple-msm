# Move passengers by air

## What the role represents

This role makes domestic passenger aviation visible as a physical non-road transport role. It is currently represented by one residual incumbent method rather than explicit aviation technologies.

## Role definition

- Role id: `move_passengers_by_air`
- Balance type: `residual_accounting`
- Output unit: `residual_activity`
- Output quantity basis: One residual activity unit carrying the passenger aviation share of the former transport-other calibration.
- Default incumbent method id: `transport_air_passenger__residual_incumbent`

## Expansion path

Replace the residual method with explicit domestic aviation service pathways when passenger-km, jet fuel, sustainable aviation fuel, and fleet evidence is authored.
