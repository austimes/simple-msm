# Move freight by rail

## What the role represents

This role makes domestic rail freight visible as a physical non-road transport role. It is currently represented by one residual incumbent method rather than explicit rail freight technologies.

## Role definition

- Role id: `move_freight_by_rail`
- Balance type: `residual_accounting`
- Output unit: `residual_activity`
- Output quantity basis: One residual activity unit carrying the freight rail share of the former transport-other calibration.
- Default incumbent method id: `transport_rail_freight__residual_incumbent`

## Expansion path

Replace the residual method with explicit bulk freight, intermodal freight, and regional freight rail pathways when tonne-km, energy, and fleet evidence is authored.
